# Balloon Clicker – HTML5 Game

Balloon Clicker is a fun and addictive HTML5 game. Click the balloon as fast as possible to reach goals and unlock higher levels.

## Features
- Fully responsive, works on desktop and mobile
- Timer per level (configurable)
- High score tracking via localStorage
- Sound effects and fireworks animation
- Buttons: Click, Repeat Level, Start Over

## How to Play
1. Open `index.html` in a modern browser.
2. Click the balloon to inflate it.
3. Reach the goal clicks before the timer runs out.
4. Advance through levels as you succeed.

## Customization
- `main.js` — adjust timer, goal, difficulty
- `style.css` — change colors, layout
- `sounds/` — replace audio files
